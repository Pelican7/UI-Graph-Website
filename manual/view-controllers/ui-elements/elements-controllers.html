Elements Controllers

An Elements Controller is a view controller based upon Unity's UIElements system. It manages an ElementsView - a view contained in a VisualElement - and stores this view in a pair of Uxml/Uss files. When working with Unity's UIElements system, all view controllers are derived from the ElementsController class.

Note

Elements controllers are currently for internal use only.
